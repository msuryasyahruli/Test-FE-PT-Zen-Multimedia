## Run Locally

Clone the project

```bash
  git clone https://github.com/msuryasyahruli/Test-FE-PT-Zen-Multimedia.git
```

Go to the project directory

```bash
  cd Test-FE-PT-Zen-Multimedia
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
